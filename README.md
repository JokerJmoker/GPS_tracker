# GPS/GSM Трекер на базе Arduino

## Описание проекта

Данный проект представляет собой многофункциональную систему GPS/GSM трекера на платформе Arduino. Система поддерживает три режима работы: отладка, циклический трекинг и энергосберегающий режим с пробуждением по движению.

## Возможности

- **Три режима работы:**
  - `DEBUG_MODE` - режим отладки, все модули работают постоянно
  - `TRACKER_MODE` - циклическая работа (GPS → SIM → GPS)
  - `SLEEP_MODE` - энергосберегающий режим с пробуждением по движению

- **Сбор GPS данных** с парсингом координат через библиотеку TinyGPS++
- **Отправка данных** через GSM модуль (AT команды)
- **Детекция движения** для пробуждения из режима сна
- **Энергосбережение** - модули работают по очереди или отключаются в режиме сна

## Аппаратное обеспечение

| Компонент | Модель | Подключение |
|-----------|--------|-------------|
| GPS | NEO-6M / NEO-7M / NEO-8M | RX → pin 3, TX → pin 4 |
| GSM | SIM800L | RX → pin 5, TX → pin 6 |
| Датчик движения | MPU6050 | I2C (SDA→A4, SCL→A5), INT→pin 2 |
| Питание | 5V / батарея | - |

## Установка и настройка

### 1. Установка библиотек

В Arduino IDE (или PlatformIO) установите следующие библиотеки:

```
TinyGPSPlus by Mikal Hart
```

### 2. Выбор режима работы

В файле `main.cpp` раскомментируйте нужный режим:

```cpp
#define MODE_DEBUG        // Режим отладки
//#define MODE_TRACKER      // Режим трекера
//#define MODE_SLEEP        // Режим сна
```

### 3. Настройка пинов

При необходимости измените пины подключения в `main.cpp`:

```cpp
NEO gps(3, 4);      // GPS: RX=3, TX=4
SIM sim800l(5, 6);  // SIM800L: RX=5, TX=6
```

## Режимы работы

### DEBUG_MODE (Отладка)

- **Назначение:** Тестирование и отладка всех модулей
- **Поведение:** Все модули работают одновременно, AT команды циклически повторяются
- **Потребление:** Высокое (~150 мА)
- **Использование:** Разработка, проверка связи с модулями

### TRACKER_MODE (Трекер)

- **Назначение:** Циклический сбор и отправка данных
- **Поведение:** 
  1. GPS собирает координаты (до получения данных)
  2. SIM отправляет 10 AT команд (по одной каждые 3 секунды)
  3. Цикл повторяется
- **Потребление:** Среднее (~80 мА среднее)
- **Использование:** Регулярное отслеживание位置

### SLEEP_MODE (Сон)

- **Назначение:** Максимальная экономия энергии
- **Поведение:**
  1. Устройство в глубоком сне
  2. При движении MPU генерирует прерывание
  3. Пробуждение → 1 цикл сбора и отправки данных
  4. Возврат в сон
- **Потребление:** Низкое (~0.1 мА в сне + 150 мА активность)
- **Использование:** Портативные устройства на батарейках

## Использование

### Пример для DEBUG_MODE

```cpp
#include "TestManager/TestManager.h"

TestManager testManager(&gps, &sim800l, &mpu6050);

void setup() {
  Serial.begin(9600);
  testManager.begin();
}

void loop() {
  testManager.update();
  delay(10);
}
```

### Пример для TRACKER_MODE

```cpp
#include "ModeManager/TrackerController.h"

TrackerController tracker(&gps, &sim800l, &mpu6050);

void setup() {
  Serial.begin(9600);
  SystemModes::begin();
  SystemModes::setMode(OperationMode::TRACKER_MODE);
  tracker.begin();
}

void loop() {
  if (SystemModes::shouldGPSBeActive()) {
    tracker.update();
  }
  
  if (SystemModes::shouldSIMBeActive()) {
    tracker.update();
  }
  
  if (SystemModes::isTrackerCycleComplete()) {
    Serial.println("Cycle complete!");
    SystemModes::resetTrackerCycle();
    delay(1000);
  }
}
```

## AT Команды SIM800L

Последовательность команд в режиме трекера:

| № | Команда | Описание |
|---|---------|----------|
| 1 | `AT+CFUN=1` | Включение радиомодуля |
| 2 | `AT` | Проверка связи |
| 3 | `AT+CPIN?` | Проверка SIM-карты |
| 4 | `AT+CCID` | Получение ID SIM-карты |
| 5 | `AT+CREG=1` | Включение регистрации в сети |
| 6 | `AT+CREG?` | Статус регистрации |
| 7 | `AT+COPS=0` | Автопоиск оператора |
| 8 | `AT+COPS?` | Информация об операторе |
| 9 | `AT+CSQ` | Уровень сигнала |
| 10 | `AT+CBC` | Напряжение питания |

## Отладка

### Монитор порта

Скорость: **9600 бод**

### Светодиодная индикация

При смене режима встроенный светодиод мигает 5 раз.

## Требования

- **Платформа:** Arduino Uno / Nano / Mega (или совместимые)
- **Память:** ~30 KB Flash, ~2 KB RAM
- **Питание:** 5V (или 3.7V литиевая батарея для SLEEP_MODE)

## Ограничения

- SIM800L требует до 2A в пиках (при передаче данных)
- В SLEEP_MODE MPU должен быть подключён к пину прерывания (pin 2)
- GPS требует открытого неба для получения сигнала
